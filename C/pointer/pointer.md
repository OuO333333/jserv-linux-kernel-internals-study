## C/C++ 變數與指標的規則

### 1. `int a`
- **解釋**：普通整數變數。
- **示例**：
  ```c
  int a = 10;
  ```

### 2. `int *a`
- **解釋**：指向 `int` 的指針。
- **示例**：
  ```c
  int x = 10;
  int *a = &x;
  ```

### 3. `int **a`
- **解釋**：指向 `int *` 的指針，即二級指針。
- **示例**：
  ```c
  int x = 10;
  int *p = &x;
  int **a = &p;
  ```

### 4. `int a[10]`
- **解釋**：長度為 `10` 的整數數組。
- **示例**：
  ```c
  int a[10];
  ```

### 5. `int *a[10]`
- **解釋**：包含 `10` 個 `int *` 指針的數組。
- **示例**：
  ```c
  int *a[10];
  ```

### 6. `int (*a)[10]`
- **解釋**：指向包含 `10` 個 `int` 的數組的指針。
- **示例**：
  ```c
  int arr[10];
  int (*a)[10] = &arr;
  ```

### 7. `int (*a)(int)`
- **解釋**：指向返回 `int` 並接受 `int` 參數的函數的指針。
- **示例**：
  ```c
  int func(int x) { return x * 2; }
  int (*a)(int) = func;
  ```

### 8. `int (*a[10])(int)`
- **解釋**：包含 `10` 個指向返回 `int` 並接受 `int` 參數的函數的指針的數組。
- **示例**：
  ```c
  int func1(int x) { return x + 1; }
  int func2(int x) { return x + 2; }
  int (*a[10])(int) = {func1, func2};
  ```
